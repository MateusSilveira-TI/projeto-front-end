Tecnologias Utilizadas:

  HTML5 e CSS3 (Semântico e Responsivo)

  JavaScript

  Estruturação de Componentes e Páginas

  Gerenciamento de Imagens e Assets Visuais

Funcionalidades: 
  Página Inicial (Home): Vitrine de produtos em destaque e banners promocionais.

  Menu de Navegação: Cabeçalho estruturado para facilitar a transição entre categorias e a área do usuário.

  Galeria de Produtos: Sistema de exibição de imagens e opções de produtos (como cores e tamanhos).

  Design Responsivo: Adaptação do layout para diferentes tamanhos de tela (Mobile e Desktop).

Estrutura de Pastas
  A organização do projeto foi pensada para manter o código limpo e escalável:

  assets/: Imagens, ícones e recursos visuais utilizados nas páginas.

  estilos/: Arquivos de estilização responsáveis pelo visual da loja.

  js/: Lógica de interatividade das páginas (menus dinâmicos, transições de galeria).

  pages/: Arquivos HTML individuais para cada tela da aplicação.
